# NutriOne-ERP

NutriOne-ERP es un sistema de planificación de recursos empresariales (ERP) diseñado específicamente para profesionales de la nutrición. Su objetivo principal es centralizar y optimizar la gestión de pacientes, permitiendo un control eficiente de historiales clínicos, pautas alimentarias, calculos nutricionales, automatizacion de calculos nutricionales y el seguimiento del progreso nutricional de forma estructurada y segura.
## Contexto Académico
Este proyecto ha sido desarrollado para la asignatura de Gestión de Proyectos de Software 2026-1 de la Universidad del Bio-Bio.
## Estructura Tecnológica
El sistema está dividido en dos capas principales, operando bajo una arquitectura cliente-servidor:
- **Frontend (Cliente):** Desarrollado utilizando React y empaquetado con Vite. Utiliza Tailwind CSS para el diseño de la interfaz de usuario, garantizando una experiencia interactiva y responsiva.
- **Backend (Servidor):** Construido sobre Node.js utilizando el framework Express. Provee una API RESTful robusta que se encarga de la lógica de negocio y la seguridad de los datos.
- **Base de Datos:** Utiliza PostgreSQL como sistema de gestión de bases de datos relacionales, asegurando la integridad y persistencia de la información clínica y administrativa.
